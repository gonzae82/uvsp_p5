# Sistema IoT Multissensorial para Monitoramento do Estado de Saúde e Manutenção Preditiva de Fontes de Alimentação utilizando ESP32

Trabalho desenvolvido para o Projeto Integrador da UNIVESP na disciplina Projeto Integrador em Computação IV. DRP14 Turma 03 - Grupo 08.

## Acessível em:
Site: https://p5.elido.com.br

Github: https://github.com/gonzae82/uvsp_p5

## Descrição

O projeto foi desenvolvido como parte do Projeto Integrador em Computação V na UNIVESP. Ele visa o desenvolvimento de ferramenta para Monitoramento de Fontes de Alimentação remotamente.
